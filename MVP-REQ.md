## 1. 产品概述

### 1.1 产品名称
**BG Remover** — AI 图像背景移除工具

### 1.2 一句话描述
用户上传图片，AI 自动移除背景，支持免费下载透明背景 PNG。

### 1.3 核心价值主张
- ⚡ **快**：5 秒内完成背景移除
- 🛡️ **隐私安全**：图片不存储，处理完即销毁
- 💯 **免费**：每月 50 张免费额度（Remove.bg API）
- 🎯 **简单**：拖拽即用，无需注册即可体验

## 3. 功能需求

### 3.1 核心功能（MVP 必须）

#### F1：图片上传
- 支持**拖拽上传**和**点击上传**两种方式
- 支持格式：JPG、JPEG、PNG、WebP
- 文件大小限制：10MB
- 上传时显示loading状态

#### F2：AI 背景移除
- 调用 Remove.bg API 完成背景移除
- **不存储**：图片在内存中处理，不落盘，无持久化存储
- 处理失败时显示友好错误提示

#### F3：结果预览
- **前后对比滑块**：拖动查看原图 / 去背景结果
- 原图 + 结果图并列展示

#### F4：下载功能
- 下载格式：PNG（透明背景）
- 一键下载到本地
- 显示文件大小

### 3.2 辅助功能（可选，后续迭代）

- [ ] 批量处理多张图片
- [ ] 支持 JPG 下载（白色背景）
- [ ] API 接口开放给第三方
- [ ] 用户体系 + 积分系统
- [ ] 付费购买额外额度

## 5. 技术方案

### 5.1 技术栈

| 层级 | 技术选型 | 说明 |
|---|---|---|
| 前端框架 | Next.js 14 + TypeScript | App Router |
| UI 样式 | Tailwind CSS + shadcn/ui | 组件库 |
| AI 处理 | Remove.bg API | 付费 API，免费 50 张/月 |
| 部署平台 | Cloudflare Pages | 免费 CDN 加速 |
| 图片处理 | 纯内存，不落盘 | 隐私安全 |

### 5.2 架构特点

- **无状态**：图片不存储，处理完即释放内存
- **Serverless**：Next.js API Routes 承接图片处理逻辑
- **边缘计算**：部署在 Cloudflare 全球边缘节点，低延迟

### 5.3 API 设计

```
POST /api/remove-bg
Request:
  Content-Type: multipart/form-data
  Body: { image: File }

Response (success):
  200 OK
  Content-Type: application/json
  { imageBase64: "data:image/png;base64,..." }

Response (error):
  400 | 500
  { error: "错误描述" }
```

### 5.4 环境变量

```env
REMOVE_BG_API_KEY=your_remove.bg_api_key
```

## 6. 已知限制

| 限制项 | 说明 | 解决方案 |
|---|---|---|
| 免费额度 50 张/月 | Remove.bg 免费套餐限制 | 后续考虑付费或自托管 RMBG-1.4 |
| 单文件 10MB | 浏览器 + API 综合限制 | 前端做文件大小校验 |
| 复杂背景处理 | Remove.bg 对复杂场景效果一般 | 后续迭代模型 |
| 无用户体系 | MVP 暂不做登录注册 | 后续接入 Auth 系统 |
